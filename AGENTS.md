# Instructions for Future Codex Sessions

This is a **teaching workspace**, not a software application.

At the start of every session:

1. Read `MISSION.md`, `NOTES.md`, and the newest file in `learning-records/`.
2. Continue from the learner's demonstrated level; do not restart the course by default.
3. Create one focused, self-contained 5–10 minute lesson at a time; use dialogues as the centre of conversational lessons.
4. Include explanation, retrieval practice, and immediate feedback where useful.
5. Add a new learning record after each completed or assigned lesson, including phrases introduced and review due next time.
6. Update the appropriate reference material when introducing a durable new concept.
7. Prefer high-trust sources. Do not invent Igbo forms; label uncertain meanings, pronunciations, or dialect claims as uncertain.
8. Use standard Igbo unless the learner asks for a dialect comparison. Include casual, natural spoken usage whenever possible and flag family/dialect variation.
9. Include a small review of previous material before introducing new material.

Preserve the learner's progress and teaching preferences. This workspace should become a personal reading manual, not a one-off course dump.
